daShef

Dream
6 months

The digital menu for restaurants and bars streamlines the ordering process through a user-friendly application, providing guests with comprehensive information about meals, including ingredients, preparation methods, and more.

A standout feature is the ability for guests to view the estimated preparation and cooking times, as well as receive real-time updates when their meal is ready to be served.

This project had actually few applications, daShef digital menu, daShef waiter, daShef guest, daShef kitchen monitor and so.

Project role
Backend developer
Responsibility
  • Implement the server side applications
  • Writing queries and filtering the data
  • Implement new models, database tables, writing migrations scripts, etc
  • Testing the application, and writing the unit-tests and integration tests
Technologies & Tools
Python Pyramid Cornice SqlAlchemy Alembic MySql GitLab Postman

This was my first project as software engineer with the focus on the backend, and that was the moment when I choose to continue in that direction later.

On the project I worked alone, but I had great colleagues that supported when I needed the help, I still was a junior software engineer and I learned much from them.

Also, I could say it was one of my favorite applications in my long carrier.
